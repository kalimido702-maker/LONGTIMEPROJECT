# نظام MASR POS Pro - دليل المميزات الشامل

## نظرة عامة
نظام نقاط بيع متكامل مبني على Electron + React + TypeScript مع دعم كامل للغة العربية.

---

## 🛒 نقطة البيع (POS)

### شاشة البيع الرئيسية
- **عرض المنتجات بالشبكة** مع البحث السريع
- **فلترة حسب الأقسام** والتصنيفات
- **سلة مشتريات ديناميكية** مع تعديل الكميات
- **دعم الباركود** للإضافة السريعة
- **تعدد أنواع الأسعار** (جملة، تجزئة، خاص)
- **حساب الخصومات والضرائب** تلقائياً
- **اختيار العميل** مع البحث

### طرق الدفع
- **نقدي** - مع حساب الباقي
- **آجل** - للعملاء الموثوقين
- **تقسيط** - مع جدولة الأقساط
- **تعدد طرق الدفع** في فاتورة واحدة (كاش + كريديت كارد)

### الطباعة
- **طباعة فاتورة حرارية** (58mm/80mm)
- **طباعة باركود المنتجات** مع السعر
- **دعم طابعات Xprinter**

## 📦 إدارة المخزون

### المنتجات
- **إضافة وتعديل المنتجات** مع الصور
- **الباركود** (EAN13, CODE128)
- **تعدد الوحدات** لكل منتج (قطعة، كرتونة، درزن)
- **تعدد الأسعار** حسب نوع العميل
- **تتبع الكميات** والتنبيه عند النفاد
- **طباعة باركود جماعية** لعدة منتجات

### التصنيفات والوحدات
- **أقسام المنتجات** مع الألوان
- **وحدات القياس** مع معامل التحويل
- **أنواع التسعير** (جملة، تجزئة، VIP)
