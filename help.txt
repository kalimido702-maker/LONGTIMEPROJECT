تمام… الكود بتاعك شغال “أساسًا”، بس فيه كام نقطة هما السبب في صداع **الأحجام** + طباعة الـ **labels / barcodes** بشكل غير ثابت، خصوصًا لما يبقى عندك طابعات 58mm و80mm (أو أكتر من طابعة USB).

هقسمهالك: (1) فين الغلط/السبب، (2) حل عملي يخلي المقاسات “تتظبط تلقائيًا” قدر الإمكان، مع كود جاهز.

---

## 1) أهم المشاكل في الكود الحالي

### A) أنت بتـعمل detect للطابعات… وبعدين بتطبع على “أول طابعة”!

انت بتجيب `printers = getUSBPrinters()`، لكن وقت الطباعة بتعمل:

```ts
const device = new escpos.USB();
```

وده معناه: **الـ USB adapter غالبًا هيختار أول طابعة لقاها** (مش اللي أنت لستها أو اللي المستخدم اختارها). وده مذكور في توثيق `escpos` إن الـ USB adapter افتراضيًا بياخد “first printer found”، والحل إنك تحدد `vid/pid` صراحة. ([npm][1])

### B) مفيش “اكتشاف موحد” لعرض الورق في ESC/POS

الـ ESC/POS عمومًا **مش بيدي API قياسي تقولك الطابعة 58 ولا 80**. فـ “التلقائي 100%” لازم يبقى:

* يا إمّا **mapping** حسب موديل/VID/PID (مش مضمون لكل الأجهزة)
* يا إمّا **إعداد مرة واحدة لكل طابعة** (وده الأكثر استقرارًا)

### C) الفواصل والأعمدة Hard-coded

عندك:

* `--------------------------------` طولها ثابت
* اسم الصنف `substring(0,16)` ثابت
* هيدر أصناف ثابت

ده هيبوّظ فورًا لما `cpl` (عدد الحروف في السطر) يختلف بين 58/80.

### D) في نقطة مهمّة جدًا: API الباركود بتختلف حسب نسخة escpos

* في `escpos@2.4.5` (القديمة)، الباركود signature بالشكل ده:
  `barcode(code, type, width, height, position, font)` ([npm][1])
* في بعض نسخ node-escpos الأحدث، بقت تقبل options object زي `{width,height,...}` ([GitHub][2])

وأنت حاليًا مستخدم:

```ts
.barcode(barcode, 'CODE39', { width: 2, height: 50 })
```

لو نسختك قديمة → ده ممكن يبوّظ الطباعة/ميطبعش باركود أصلاً.

### E) على Windows: USB Direct محتاج Driver (WinUSB)

لو بتطبع direct عبر `node-usb`/`escpos-usb` على ويندوز، ساعات لازم تستخدم **Zadig** عشان تثبّت WinUSB للطابعة، وإلا ممكن يجيلك `LIBUSB_ERROR_NOT_SUPPORTED`. ([npm][1])

---

## 2) الحل العملي اللي “يظبط الأحجام تلقائيًا” بجد

### الفكرة

1. **لازم تختار طابعة بعينها**: باستخدام `(vendorId, productId)` وتفتحها:
   `new escpos.USB(vid, pid)` ([npm][1])
2. تعمل **Printer Profile** لكل طابعة:

   * `paper: "58mm" | "80mm"`
   * `cpl: 32 | 48` (أو 42/64 حسب الموديل)
   * `encoding` (لو هتطبع عربي كنص)
   * `cutter: boolean`
3. تستخدم `cpl` ده عشان:

   * تبني الفواصل: `"-".repeat(cpl)`
   * تحسب أعمدة الأصناف ديناميكيًا
4. للباركود: اعمل Wrapper يدعم النسختين (object options / positional args).

> **مهم**: “تلقائي بالكامل” من غير إعدادات مش مضمون لأن ESC/POS مش بيقولك عرض الورق بشكل قياسي. أقوى حل:
> **أول مرة تشوف طابعة جديدة**: تخلي المستخدم يختار 58 ولا 80 (مرة واحدة وتتخزن)، وبعدها كل حاجة تبقى تلقائي.

## 3) كود جاهز (Drop-in) لتحسين طباعة الفاتورة + الباركود

### 3.1 Helpers للعرض (cpl) وتنسيق الأعمدة

```ts
type PaperKind = '58mm' | '80mm';

type UsbTarget = {
  vendorId: number;
  productId: number;
};

type PrinterProfile = {
  target: UsbTarget;
  paper: PaperKind;
  cpl?: number;           // chars per line (لو مش متسجل هيتحسب من paper)
  encoding?: string;      // مثال: 'CP864' أو 'windows1256' حسب دعم الطابعة
  cutter?: boolean;       // لو الطابعة فيها cutter
};

const PAPER_CPL: Record<PaperKind, number> = {
  '58mm': 32,
  '80mm': 48,
};

function getCpl(profile: PrinterProfile): number {
  return profile.cpl ?? PAPER_CPL[profile.paper];
}

function hr(cpl: number, ch = '-'): string {
  return ch.repeat(Math.max(1, cpl));
}

function truncate(s: string, max: number): string {
  if (max <= 0) return '';
  return s.length > max ? s.slice(0, Math.max(0, max - 1)) + '…' : s;
}

function pad(
  text: string,
  width: number,
  align: 'lt' | 'ct' | 'rt' = 'lt'
): string {
  const t = truncate(text ?? '', width);
  const space = Math.max(0, width - t.length);
  if (align === 'rt') return ' '.repeat(space) + t;
  if (align === 'ct') {
    const left = Math.floor(space / 2);
    const right = space - left;
    return ' '.repeat(left) + t + ' '.repeat(right);
  }
  return t + ' '.repeat(space);
}

function row3(
  cpl: number,
  col1: { text: string; width: number; align?: 'lt' | 'ct' | 'rt' },
  col2: { text: string; width: number; align?: 'lt' | 'ct' | 'rt' },
  col3: { text: string; width: number; align?: 'lt' | 'ct' | 'rt' }
): string {
  const totalW = col1.width + col2.width + col3.width;
  // حماية بسيطة لو حصل mismatch
  if (totalW !== cpl) {
    // نوزع الفرق على العمود الأول (الأسماء)
    col1 = { ...col1, width: col1.width + (cpl - totalW) };
  }
  return (
    pad(col1.text, col1.width, col1.align ?? 'lt') +
    pad(col2.text, col2.width, col2.align ?? 'rt') +
    pad(col3.text, col3.width, col3.align ?? 'rt')
  );
}
```

### 3.2 اختيار الطابعة الصح (VID/PID) بدل “أول طابعة”

```ts
function createUsbDevice(escpos: any, profile: PrinterProfile) {
  // ده مهم جدًا: تحديد VID/PID عشان متطبعش على أول طابعة
  return new escpos.USB(profile.target.vendorId, profile.target.productId);
}
```

> توثيق `escpos` بيوضح إنك تقدر تعمل `USB(vid,pid)` وإن الافتراضي بيختار أول طابعة ([npm][1])

### 3.3 Wrapper للـ barcode (يدعم نسختين)

```ts
function printBarcodeCompat(
  printer: any,
  code: string,
  type: string,
  opts: { width?: number; height?: number; position?: string; font?: string } = {}
) {
  // نحاول API الحديثة (object options)
  try {
    printer.barcode(code, type, { width: opts.width ?? 2, height: opts.height ?? 80 });
    return;
  } catch {}

  // fallback لنسخ escpos القديمة (positional args)
  // في escpos@2.4.5: barcode(code, type, width, height, position, font) :contentReference[oaicite:6]{index=6}
  const width = opts.width ?? 64;     // ملحوظة: المعنى بيختلف بين الإصدارات
  const height = opts.height ?? 3;
  const position = (opts.position ?? 'BELOW').toUpperCase();
  const font = (opts.font ?? 'A').toUpperCase();
  printer.barcode(code, type, width, height, position, font);
}
```

### 3.4 طباعة الفاتورة بـ cpl ديناميكي

> هنا الفكرة: نحسب عرض الأعمدة حسب `cpl` بدل الأرقام الثابتة.

```ts
async function printInvoiceReceiptV2(
  escpos: any,
  invoiceData: any,
  profile: PrinterProfile
): Promise<{ success: boolean; error?: string }> {
  return new Promise((resolve) => {
    const cpl = getCpl(profile);
    const line = hr(cpl, '-');
    const line2 = hr(cpl, '=');

    // أعمدة الأصناف
    const qtyW = 6;       // "  12x" أو "  9999"
    const totalW = 12;    // "  12345.67"
    const nameW = cpl - qtyW - totalW;

    const device = createUsbDevice(escpos, profile);
    const printer = new escpos.Printer(device);

    device.open((err: any) => {
      if (err) {
        resolve({ success: false, error: 'فشل فتح الطابعة: ' + (err.message ?? err) });
        return;
      }

      try {
        // Encoding (اختياري) - escpos بيستخدم iconv-lite :contentReference[oaicite:7]{index=7}
        if (profile.encoding) {
          printer.encode(profile.encoding);
        }

        printer
          .font('a')
          .align('ct')
          .style('b')
          .size(1, 1)
          .text(invoiceData.storeName)
          .size(0, 0)
          .style('normal');

        if (invoiceData.storeAddress) printer.text(invoiceData.storeAddress);
        if (invoiceData.storePhone) printer.text('Tel: ' + invoiceData.storePhone);
        if (invoiceData.taxNumber) printer.text('Tax: ' + invoiceData.taxNumber);

        printer.text(line);

        const title =
          invoiceData.paymentType === 'cash'
            ? 'فاتورة نقدية'
            : invoiceData.paymentType === 'credit'
              ? 'فاتورة آجلة'
              : 'فاتورة تقسيط';

        printer.align('ct').style('b').text(title).style('normal');
        printer.text(line);

        // Invoice info
        printer
          .align('lt')
          .text('رقم: ' + invoiceData.invoiceId)
          .text('التاريخ: ' + invoiceData.date)
          .text('الوقت: ' + invoiceData.time);

        if (invoiceData.customerName) printer.text('العميل: ' + invoiceData.customerName);
        if (invoiceData.employeeName) printer.text('البائع: ' + invoiceData.employeeName);

        printer.text(line);

        // Header row
        printer.text(
          row3(
            cpl,
            { text: 'الصنف', width: nameW, align: 'lt' },
            { text: 'الكمية', width: qtyW, align: 'rt' },
            { text: 'الإجمالي', width: totalW, align: 'rt' }
          )
        );
        printer.text(line);

        // Items
        for (const item of invoiceData.items ?? []) {
          printer.text(
            row3(
              cpl,
              { text: item.name ?? '', width: nameW, align: 'lt' },
              { text: String(item.quantity ?? ''), width: qtyW, align: 'rt' },
              { text: Number(item.total ?? 0).toFixed(2), width: totalW, align: 'rt' }
            )
          );
        }

        printer.text(line);

        // Totals (يمين)
        const money = (v: number) => `${Number(v ?? 0).toFixed(2)} ${invoiceData.currency}`;

        printer.align('lt');
        // نعرضها كسطور “Label : Value” بس من غير ما تكسر السطر
        printer.text(row3(cpl,
          { text: 'المجموع', width: cpl - totalW, align: 'lt' },
          { text: '', width: 0, align: 'lt' },
          { text: money(invoiceData.subtotal), width: totalW, align: 'rt' }
        ));

        if (invoiceData.discount && invoiceData.discount > 0) {
          printer.text(row3(cpl,
            { text: 'الخصم', width: cpl - totalW, align: 'lt' },
            { text: '', width: 0, align: 'lt' },
            { text: '-' + money(invoiceData.discount), width: totalW, align: 'rt' }
          ));
        }

        if (invoiceData.tax && invoiceData.tax > 0) {
          printer.text(row3(cpl,
            { text: 'الضريبة', width: cpl - totalW, align: 'lt' },
            { text: '', width: 0, align: 'lt' },
            { text: money(invoiceData.tax), width: totalW, align: 'rt' }
          ));
        }

        printer.text(line2);
        printer.style('b').size(1, 1);
        printer.text(`الإجمالي: ${money(invoiceData.total)}`);
        printer.size(0, 0).style('normal');
        printer.text(line2);

        printer.text(`المدفوع: ${money(invoiceData.paidAmount)}`);

        if (invoiceData.remainingAmount && invoiceData.remainingAmount > 0) {
          printer.text(`المتبقي: ${money(invoiceData.remainingAmount)}`);
        } else if (invoiceData.paidAmount > invoiceData.total) {
          const change = invoiceData.paidAmount - invoiceData.total;
          printer.text(`الباقي: ${money(change)}`);
        }

        printer.text(line);

        printer
          .align('ct')
          .text('شكراً لزيارتكم')
          .text('نتشرف بخدمتكم دائماً')
          .feed(3);

        // cut اختياري
        if (profile.cutter !== false) {
          printer.cut();
        }

        printer.close(() => resolve({ success: true }));
      } catch (e: any) {
        resolve({ success: false, error: 'خطأ في الطباعة: ' + (e.message ?? e) });
      }
    });
  });
}
```

### 3.5 طباعة Barcode Label “تظبط نفسها” حسب cpl + توافق API

```ts
async function printBarcodeLabelV2(
  escpos: any,
  labelData: any,
  profile: PrinterProfile
): Promise<{ success: boolean; error?: string }> {
  return new Promise((resolve) => {
    const cpl = getCpl(profile);
    const device = createUsbDevice(escpos, profile);
    const printer = new escpos.Printer(device);

    device.open((err: any) => {
      if (err) {
        resolve({ success: false, error: 'فشل فتح الطابعة: ' + (err.message ?? err) });
        return;
      }

      try {
        if (profile.encoding) printer.encode(profile.encoding);

        const copies = labelData.copies || 1;

        for (let i = 0; i < copies; i++) {
          printer.align('ct');

          // اسم المنتج يتقص على قد السطر (عشان 58/80)
          printer.text(truncate(labelData.productName ?? '', cpl));

          // barcode
          // ملاحظة: النوع المناسب غالبًا CODE128 لو البيانات متنوعة
          printBarcodeCompat(printer, String(labelData.barcode ?? ''), 'CODE39', {
            width: profile.paper === '80mm' ? 3 : 2,
            height: 80,
          });

          printer.text(String(labelData.barcode ?? ''));

          if (labelData.price !== undefined) {
            const cur = labelData.currency || 'EGP';
            printer.style('b').text(`${Number(labelData.price).toFixed(2)} ${cur}`).style('normal');
          }

          printer.feed(1);
        }

        if (profile.cutter !== false) printer.cut();
        printer.close(() => resolve({ success: true }));
      } catch (e: any) {
        resolve({ success: false, error: 'خطأ في الطباعة: ' + (e.message ?? e) });
      }
    });
  });
}
```

## 4) تعديل الـ IPC عندك (الأهم: ابعت printer profile مع كل print)

بدل ما `print-invoice` ياخد `invoiceData` بس… خليه ياخد كمان `printerProfile` (أو أقل حاجة vendorId/productId + paper).

مثال:

```ts
ipcMain.handle('escpos:print-invoice', async (_event, { invoiceData, printerProfile }) => {
  await initEscpos();
  return await printInvoiceReceiptV2(escpos, invoiceData, printerProfile);
});

ipcMain.handle('escpos:print-barcode', async (_event, { labelData, printerProfile }) => {
  await initEscpos();
  return await printBarcodeLabelV2(escpos, labelData, printerProfile);
});
```

وفي `get-printers` رجّع VID/PID زي ما بتعمل (ده الأساس)، بس خزن profile لكل واحد في settings.

## 5) ملاحظة مهمّة عن العربي (لو عندك حروف بتطلع مفصولة/مقلوبة)

`escpos` بيعتمد على iconv-lite للـ encoding، والـ ESC/POS مش مضمون يدعم UTF‑8 للعربي على كل الطابعات. ([npm][1])
لو العربي عندك “مش مظبوط” كحروف: الحل الأكثر ضمانًا هو **طباعة Raster Image** (ترسم الفاتورة/الليبل كصورة PNG من Canvas/HTML وتبعتها `printer.image(...)`) — ده بيحل:

* العربي (تشكيل/RTL)
* اختلاف المقاسات
* أي Font/Logo/Barcode مضبوط 100%

(لو تحب أمشي معاك في خيار الـ raster ده برضه، هتطلع بنتيجة production-grade جدًا.)

## 6) Checklist سريع لو بتجرب على Windows وبتشوف Errors

لو بتطلع لك مشاكل فتح الـ USB أو `LIBUSB_ERROR_NOT_SUPPORTED`:

* ده غالبًا محتاج WinUSB عبر Zadig للطابعة ([npm][1])

### الخلاصة

* **أكبر Bug عندك**: بتـdetect الطابعات لكن بتطبع على `new escpos.USB()` بدون تحديد VID/PID → فتطبع على أول طابعة ومقاسات غلط. ([npm][1])
* **حل المقاسات**: اعمل `PrinterProfile` لكل طابعة + `cpl` ديناميكي + فواصل/أعمدة محسوبة.
* **الـ labels والـ barcodes**: اعمل wrapper للـ barcode عشان اختلاف نسخ المكتبة. ([npm][1])

لو طبّقت الهيكل ده، موضوع “الأحجام تتحل تلقائي” هيبقى عملي جدًا:
**مرة واحدة بس** تختار 58/80 لكل طابعة (أو تعمل mapping لو عندك موديلات محددة)، وبعدها كل الطباعة تبقى auto.
