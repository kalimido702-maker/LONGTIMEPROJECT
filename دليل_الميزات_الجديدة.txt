================================================================================
                     دليل الميزات الكامل - نظام نقاط البيع
================================================================================

هذا الدليل يشرح جميع الميزات الجديدة والمحسّنة في النظام.


================================================================================
                          1. خيارات التسليم في الفواتير
================================================================================

الوصف:
------
تتبع حالة تسليم الطلبات للعملاء مع إمكانية تحديث الحالة من نقطة البيع 
أو من صفحة الفواتير.

الحالات المتاحة:
----------------
• لم يتم التسليم - (الحالة الافتراضية عند إنشاء الفاتورة)
• تم الشحن - (عند إرسال الطلب للتوصيل)
• تم التسليم - (عند استلام العميل للطلب)

طريقة الاستخدام:
-----------------
1. عند إنشاء فاتورة جديدة:
   - في نقطة البيع، ستجد قائمة "حالة التسليم" في الجزء الأيمن
   - اختر الحالة المناسبة قبل حفظ الفاتورة

2. تعديل حالة التسليم من صفحة الفواتير:
   - افتح صفحة "الفواتير" من القائمة
   - في عمود "التسليم"، اضغط على القائمة المنسدلة
   - غيّر الحالة مباشرة

3. تصفية الفواتير حسب حالة التسليم:
   - في صفحة الفواتير، استخدم فلتر "حالة التسليم"

الفوائد:
--------
✓ تتبع دقيق لحالة الطلبات
✓ سهولة معرفة الطلبات المعلقة للتوصيل
✓ تحسين خدمة العملاء والمتابعة


================================================================================
                          2. صفحة القبض السريع
================================================================================

الوصف:
------
صفحة مخصصة لتسجيل المبالغ المقبوضة من العملاء بشكل سريع وفوري.

طريقة الوصول:
--------------
من القائمة ← المالية ← القبض السريع

طريقة الاستخدام:
-----------------
1. ابحث عن العميل واختره (سيظهر رصيده تلقائياً)
2. أدخل المبلغ أو اضغط "كامل الرصيد" أو "نصف الرصيد"
3. اختر طريقة الدفع
4. اضغط "قبض" لتسجيل العملية

الفوائد:
--------
✓ سرعة تسجيل المقبوضات دون الحاجة لفتح فواتير
✓ رؤية فورية لإجمالي المديونية
✓ تحديث رصيد العميل تلقائياً


================================================================================
                          3. نظام المخازن المتعددة
================================================================================

الوصف:
------
إمكانية إدارة أكثر من مخزن (فرع/مستودع) واختيار المخزن عند البيع.

طريقة الوصول لإدارة المخازن:
-----------------------------
من القائمة ← الإعدادات الأساسية ← المخازن

طريقة الاستخدام:
-----------------
1. أضف المخازن المطلوبة
2. عيّن مخزن افتراضي
3. في نقطة البيع، اختر المخزن المراد الخصم منه (إذا كان أكثر من واحد)

ملاحظة: قائمة المخزن تظهر فقط عند وجود أكثر من مخزن نشط.

الفوائد:
--------
✓ إدارة مخازن/فروع متعددة
✓ معرفة من أي مخزن تم البيع
✓ مرونة في التوسع


================================================================================
                  4. هيكلة المستخدمين (مشرفين - مناديب - عملاء)
================================================================================

الوصف:
------
نظام ثلاثي المستويات: المشرفين > المناديب > العملاء

طريقة الوصول:
-------------
• المشرفين: الإدارة ← المشرفين
• المناديب: الإدارة ← المندوبين

إعداد الهيكل:
--------------
1. أضف المشرفين أولاً (الإدارة ← المشرفين)
2. أضف المناديب واربط كل مندوب بمشرفه
3. عند إضافة عميل، اختر المندوب المسؤول عنه

طريقة الاستخدام:
-----------------
• كل مندوب مرتبط بمشرف
• كل عميل مرتبط بمندوب
• في التقارير، يمكن الفلترة حسب المشرف أو المندوب

الفوائد:
--------
✓ تتبع أداء كل مندوب
✓ معرفة عملاء كل مندوب
✓ تقارير حسب المشرف/المندوب


================================================================================
                          5. فلاتر التقارير المتقدمة
================================================================================

الوصف:
------
فلاتر متقدمة في التقارير لتحديد الفترة والمشرف والمندوب والتصنيف.

طريقة الوصول:
-------------
من صفحة التقارير ← الفلاتر المتقدمة

الفلاتر المتاحة:
----------------
• الفترة الزمنية (من - إلى)
• المشرف
• المندوب (يتغير حسب المشرف المحدد)
• تصنيف المنتجات

الفوائد:
--------
✓ تقارير دقيقة ومحددة
✓ تحليل أداء كل مندوب/مشرف
✓ مقارنة فترات زمنية مختلفة


================================================================================
                          6. تصدير البيانات إلى Excel
================================================================================

الوصف:
------
تصدير التقارير والبيانات إلى ملفات Excel.

طريقة الاستخدام:
-----------------
في معظم الصفحات (العملاء، الفواتير، التقارير...) 
ابحث عن زر "تصدير Excel" واضغط عليه.

الفوائد:
--------
✓ سهولة تحليل البيانات خارجياً
✓ مشاركة التقارير مع الآخرين
✓ أرشفة البيانات


================================================================================
                          7. إضافة ملاحظات للفاتورة
================================================================================

الوصف:
------
إمكانية إضافة ملاحظة نصية لكل فاتورة.

طريقة الاستخدام:
-----------------
في نقطة البيع، قبل حفظ الفاتورة:
- اضغط على أيقونة الملاحظات أو مربع النص
- اكتب الملاحظة المطلوبة
- احفظ الفاتورة

الفوائد:
--------
✓ توثيق تفاصيل إضافية للفاتورة
✓ تعليمات خاصة للتوصيل
✓ معلومات للمتابعة


================================================================================
                          8. العدد في الكرتونة
================================================================================

الوصف:
------
حقل "العدد في الكرتونة" للمنتجات لعرضه في الفاتورة.

طريقة الإعداد:
---------------
في المخزون ← تعديل المنتج ← أدخل "العدد في الكرتونة"

طريقة الظهور:
--------------
• يظهر في نقطة البيع عند إدخال الكمية
• يُطبع في الفاتورة

الفوائد:
--------
✓ تسهيل حساب الكميات للعاملين
✓ وضوح أكبر للعميل في الفاتورة


================================================================================
                   9. نظام الوحدات المتعددة (كرتونة/علبة/قطعة)
================================================================================

الوصف:
------
بيع المنتج بوحدات مختلفة (كرتونة، علبة، قطعة) مع تحويل تلقائي للمخزون.

طريقة الإعداد:
---------------
1. المخزون ← اختر المنتج ← "إدارة الوحدات"
2. أضف وحدة جديدة (كرتونة) مع معامل التحويل (مثلاً: كرتونة = 12 قطعة)
3. حفظ

طريقة الاستخدام:
-----------------
في نقطة البيع:
- عند اختيار المنتج، اختر الوحدة المطلوبة (قطعة أو كرتونة)
- المخزون يُخصم تلقائياً بالكمية الصحيحة

الفوائد:
--------
✓ مرونة في البيع (جملة/تجزئة)
✓ دقة في حساب المخزون
✓ سعر مختلف لكل وحدة


================================================================================
                          10. إرسال الفواتير عبر WhatsApp
================================================================================

الوصف:
------
إرسال الفواتير للعملاء أو المناديب عبر WhatsApp.

طريقة الاستخدام:
-----------------
بعد حفظ الفاتورة:
1. ستظهر نافذة لاختيار المستلم
2. اختر: العميل فقط / المندوب فقط / كلاهما
3. اضغط إرسال

الفوائد:
--------
✓ إرسال فوري للفاتورة
✓ توثيق إلكتروني
✓ تحسين تجربة العميل


================================================================================
                          11. نظام الصلاحيات والأدوار
================================================================================

الوصف:
------
تحديد صلاحيات كل مستخدم بشكل دقيق.

طريقة الوصول:
-------------
النظام ← الأدوار والصلاحيات

طريقة الإعداد:
---------------
1. أنشئ دور جديد (مثلاً: "مشاهد فقط")
2. حدد الصلاحيات لكل قسم (عرض/إضافة/تعديل/حذف)
3. عين الدور للمستخدم المطلوب

الفوائد:
--------
✓ أمان أفضل للبيانات
✓ كل مستخدم يرى ما يحتاجه فقط
✓ منع التعديلات غير المرغوبة


================================================================================
                   12. حملات WhatsApp وإرسال كشوف الحسابات
================================================================================

الوصف:
------
إرسال رسائل جماعية وكشوف حسابات للعملاء والمناديب.

طريقة الوصول:
-------------
الواتساب ← الحملات التسويقية

الميزات:
---------
• إرسال لمجموعة عملاء محددة
• إرسال كشف حساب
• إرسال الرصيد الحالي فقط
• جدولة الإرسال

الفوائد:
--------
✓ تواصل فعال مع العملاء
✓ تذكير بالمديونيات
✓ حملات ترويجية
